Resumaker v0.2
Генератор html-анкет с шаблонизацией.

На вход принимает личные данные и шаблон оформления резюме
а выходе генерирует веб-страничку резюме в той же папке.


Например:

У нас есть файлы:
    - alex.properies - личные данные анкеты
    - bootstrap.cvml - вариант оформления анкеты

Нам нужен файл:
    - alex.html - оформленная анкета с данными Александра

для получения оформленной анкеты нужно в из папки с файлами программы 
нужно выполнить команды:

    ~ java -jar resumaker.jar marat.properties --template default.cvml

В той же папке будет с генерирована веб-страница alex.html
